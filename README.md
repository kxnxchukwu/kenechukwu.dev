# kenechukwu.dev

Personal portfolio website built with React, TypeScript, Vite and TailwindCSS.

## Stack

- React 19
- TypeScript
- Vite
- TailwindCSS v4
- MUI Icons

## Pages

- `/` — Home
- `/work` — Experience
- `/projects` — Projects
- `/about` — About

## Running locally

```bash
yarn install
yarn dev
```

## Deployment

Deployed on Vercel at [kenechukwu.dev](https://kenechukwu.dev)
